This engine already has: Three.js rendering, Rapier3D physics, Blueprint visual scripting, Animation Graph (state machines with blendspaces), Widget/UI system, Class inheritance, Content Browser, Project save/load (JSON), Actor/Component system, Docking panel system.
DO NOT rewrite anything that already works.

Core Goals
This prompt adds a fully functional 2D game mode to the existing engine. The key principles are:

Mode switching — the user can switch between 3D and 2D scenes; systems adapt automatically
Shared architecture — the existing Animation Blueprint, Widget system, and Blueprint node graph are reused and extended, not replaced
Sprite animation tool — a dedicated visual editor for creating sprite sheet animations, which feeds into the existing Animation Blueprint (states reference sprite animations instead of skeletal clips)
Blendspaces work the same — the existing blendspace system works in 2D scenes; nodes read velocity/input values and blend between sprite animations exactly as they do between 3D clips
Tilemap editor — a full tile painting editor with layers, collision generation, and per-tile properties
Character Pad 2D — a dedicated panel for configuring a 2D character's movement types, physics, and animation links all in one place
2D Actor system — actors with SpriteRenderer, RigidBody2D, and Collider2D components; logic is wired in the existing Blueprint graph
Widget system unchanged — existing Widget Blueprints work in 2D scenes without modification

2. Orthographic Camera (Camera2D)
Replaces PerspectiveCamera in 2D mode. Pan with middle mouse or Alt+drag. Zoom toward cursor with scroll wheel.
javascriptclass Camera2D {
  constructor(renderer, settings = {}) {
    this.pixelsPerUnit = settings.pixelsPerUnit || 100;
    this.zoom = 1.0;
    this._buildCamera();
    this._bindControls(renderer.domElement);
  }

  _buildCamera() {
    const { w, h } = this._viewSize();
    this.camera = new THREE.OrthographicCamera(-w/2, w/2, h/2, -h/2, -1000, 1000);
    this.camera.position.set(0, 0, 100);
    this.camera.lookAt(0, 0, 0);
  }

  setZoom(zoom) {
    this.zoom = Math.max(0.05, Math.min(20, zoom));
    const { w, h } = this._viewSize();
    Object.assign(this.camera, { left: -w/2, right: w/2, top: h/2, bottom: -h/2 });
    this.camera.updateProjectionMatrix();
  }

  screenToWorld(sx, sy) {
    const v = new THREE.Vector3(
      (sx / window.innerWidth) * 2 - 1,
      -((sy / window.innerHeight) * 2 - 1), 0
    );
    v.unproject(this.camera);
    return { x: v.x, y: v.y };
  }

  update() {
    if (this._followTarget) {
      const { x, y } = this._followTarget.group.position;
      const s = this._followSmoothing;
      this.camera.position.x += (x - this.camera.position.x) * s;
      this.camera.position.y += (y - this.camera.position.y) * s;
    }
  }

  follow(actor, smoothing = 0.1) {
    this._followTarget = actor;
    this._followSmoothing = smoothing;
  }
}

javascriptclass SpriteAnimationEditor {
  constructor(container) {
    this.container = container;
    this.activeSpriteSheet = null;
    this.activeAnimation = null;
    this.isPlaying = false;
    this.currentFrame = 0;
    this.frameTimer = 0;
    this.playbackSpeed = 1.0;

    this.previewCanvas = document.createElement('canvas');
    this.previewCanvas.width = 128;
    this.previewCanvas.height = 128;
    this.previewCtx = this.previewCanvas.getContext('2d');
    this.previewCtx.imageSmoothingEnabled = false;
  }

  tickPreview(timestamp) {
    if (!this.isPlaying) return;
    const dt = (timestamp - this.lastTime) / 1000;
    this.lastTime = timestamp;
    this.frameTimer += dt * this.playbackSpeed;

    const frameDuration = 1.0 / this.activeAnimation.fps;
    while (this.frameTimer >= frameDuration) {
      this.frameTimer -= frameDuration;
      this.currentFrame++;
      if (this.currentFrame >= this.activeAnimation.frames.length) {
        this.currentFrame = this.activeAnimation.loop ? 0 : this.activeAnimation.frames.length - 1;
        if (!this.activeAnimation.loop) { this.stopPreview(); return; }
      }
    }

    this.renderPreviewFrame();
    requestAnimationFrame(t => this.tickPreview(t));
  }

  renderPreviewFrame() {
    const frameId = this.activeAnimation.frames[this.currentFrame];
    const sprite = this.activeSpriteSheet.sprites.find(s => s.spriteId === frameId);
    if (!sprite) return;

    const ctx = this.previewCtx;
    ctx.clearRect(0, 0, 128, 128);
    this.drawCheckerboard(ctx, 128, 128);

    const scale = Math.min(128 / sprite.width, 128 / sprite.height);
    const dw = sprite.width * scale;
    const dh = sprite.height * scale;
    ctx.drawImage(
      this.activeSpriteSheet.image,
      sprite.x, sprite.y, sprite.width, sprite.height,
      (128 - dw) / 2, (128 - dh) / 2, dw, dh
    );
  }
}

4. Animation Blueprint — 2D Extension
The existing Animation Blueprint system is reused exactly. The only difference: states reference animation2D (a sprite animation ID from a sprite sheet) instead of a skeletal clip.
Blendspaces Work the Same
Existing blendspace nodes work identically in 2D. Instead of blending between skeletal poses, they cross-fade between sprite animations or select which animation to play based on a 1D/2D parameter (e.g. speed, direction). The variable system, condition evaluator, and transition graph are unchanged.

2D → 3D Compatibility

In a 3D scene: Animation Blueprint states reference skeletal animation clips → existing system
In a 2D scene: Animation Blueprint states reference sprite animation IDs → SpriteAnimator drives UV offsets on the sprite mesh

The graph editor, variable panel, condition editor, and blendspace canvas are the same UI in both modes. The runtime executor checks sceneMode to decide whether to drive a skeletal mesh or a sprite UV.
SpriteAnimator Runtime
javascriptclass SpriteAnimator {
  constructor(actor, animBlueprintId) {
    this.actor = actor;
    this.blueprint = assetManager.get(animBlueprintId);
    this.spriteSheet = assetManager.get(this.blueprint.spriteSheet);
    this.spriteRenderer = actor.getComponent('SpriteRenderer');
    this.variables = {};
    this.currentStateId = null;
    this.currentAnim = null;
    this.currentFrame = 0;
    this.frameTimer = 0;
    this._init();
  }

  update(deltaTime) {
    this._syncAutoVariables();   // speed, isGrounded, velocityX, velocityY
    this._evaluateTransitions(); // same logic as 3D — reads condition strings
    this._advanceFrame(deltaTime);
  }

  _syncAutoVariables() {
    const rb = this.actor.getComponent('RigidBody2D');
    if (!rb?.rigidBody) return;
    const vel = rb.rigidBody.linvel();
    this.variables.speed = Math.abs(vel.x);
    this.variables.velocityX = vel.x;
    this.variables.velocityY = vel.y;
    this.variables.isGrounded = rb.isGrounded;
    this.variables.isJumping = vel.y > 50 && !rb.isGrounded;
    this.variables.isFalling = vel.y < -50 && !rb.isGrounded;
  }

  _applyFrame() {
    const frameId = this.currentAnim.frames[this.currentFrame];
    const sprite = this.spriteSheet.sprites.find(s => s.spriteId === frameId);
    if (sprite) this.spriteRenderer.setSprite(sprite, this.spriteSheet.texture);
  }

  _checkEvents(frameIndex) {
    (this.currentAnim.events || [])
      .filter(e => e.frame === frameIndex)
      .forEach(e => this.actor.emit('animEvent_' + e.name));
  }

  setVariable(name, value) { this.variables[name] = value; }
  play(animName) {
    const anim = this.spriteSheet.animations.find(a => a.animName === animName);
    if (anim && anim.animId !== this.currentAnim?.animId) {
      this.currentAnim = anim;
      this.currentFrame = 0;
      this.frameTimer = 0;
    }
  }
}

javascriptclass TilePaintTool {
  paint(cell) {
    const stamp = this.editor.palette.selected;  // multi-tile stamp
    const layer = this.editor.activeLayer;

    stamp.forEach(({ x: sx, y: sy }) => {
      const tx = cell.x + sx - stamp[0].x;
      const ty = cell.y + sy - stamp[0].y;
      const tileId = this.editor.activeTileset.getTileId(sx, sy);
      const key = `${tx},${ty}`;
      this.undoBatch.push({ key, prev: layer.tiles[key], next: tileId });
      layer.tiles[key] = tileId;
    });

    // Rebuild Three.js mesh for affected cells only (not full layer)
    this.editor.activeTilemap.rebuildCells(layer, stamp.map((_, i) => ({
      x: cell.x + stamp[i].x - stamp[0].x,
      y: cell.y + stamp[i].y - stamp[0].y,
    })));

    // Rebuild collision — debounced 200ms
    this.editor.scheduleCollisionRebuild(layer);
  }

  fill(startCell) {
    // BFS flood fill — only replaces matching tile ID
    const layer = this.editor.activeLayer;
    const targetId = layer.tiles[`${startCell.x},${startCell.y}`] ?? null;
    const fillId = this.editor.palette.selected[0];
    if (targetId === fillId) return;

    const queue = [startCell];
    const visited = new Set();
    while (queue.length > 0) {
      const { x, y } = queue.shift();
      const key = `${x},${y}`;
      if (visited.has(key)) continue;
      visited.add(key);
      if ((layer.tiles[key] ?? null) !== targetId) continue;
      this.undoBatch.push({ key, prev: layer.tiles[key], next: fillId });
      layer.tiles[key] = fillId;
      queue.push({ x: x+1, y }, { x: x-1, y }, { x, y: y+1 }, { x, y: y-1 });
    }

    this.editor.activeTilemap.rebuildLayer(layer);
    this.editor.scheduleCollisionRebuild(layer);
  }
}
Auto-Collision Generation
Merges adjacent solid tiles into as few Rapier2D static bodies as possible (greedy rectangle merge), then rebuilds on every paint operation (debounced 200ms):
javascriptclass TilemapCollisionBuilder {
  rebuild(layer, physics2DWorld, tileset) {
    physics2DWorld.removeLayerBodies(layer.layerId);
    if (!layer.hasCollision) return;

    const merged = this.mergeRects(layer.tiles, tileset);
    merged.forEach(rect => {
      const w = rect.cols * (tileset.tileWidth / pixelsPerUnit);
      const h = rect.rows * (tileset.tileHeight / pixelsPerUnit);
      const cx = rect.x * (tileset.tileWidth / pixelsPerUnit) + w / 2;
      const cy = rect.y * (tileset.tileHeight / pixelsPerUnit) + h / 2;
      physics2DWorld.addStaticBox(layer.layerId, cx, cy, w, h);
    });
  }
}

javascriptclass CharacterMovement2D {
  update(deltaTime) {
    const rb = this.actor.getComponent('RigidBody2D');
    if (!rb) return;

    this.isGrounded = rb.isGrounded;
    if (this.wasGrounded && !this.isGrounded) this.coyoteTimer = this.properties.coyoteTime;
    if (this.coyoteTimer > 0) this.coyoteTimer -= deltaTime;
    if (this.jumpBufferTimer > 0) {
      this.jumpBufferTimer -= deltaTime;
      if (this.isGrounded || this.coyoteTimer > 0) this._doJump(rb);
    }
    if (this.isGrounded && !this.wasGrounded) this.jumpsRemaining = this.properties.maxJumps;

    const vel = rb.rigidBody.linvel();
    if (vel.y < this.properties.maxFallSpeed)
      rb.rigidBody.setLinvel({ x: vel.x, y: this.properties.maxFallSpeed }, true);

    this.wasGrounded = this.isGrounded;
  }

  moveHorizontal(direction, deltaTime, isRunning = false) {
    const rb = this.actor.getComponent('RigidBody2D');
    const targetSpeed = direction * (isRunning ? this.properties.runSpeed : this.properties.moveSpeed);
    const accel = this.properties.acceleration * (this.isGrounded ? 1.0 : this.properties.airControl);
    const vel = rb.rigidBody.linvel();
    rb.rigidBody.setLinvel({
      x: THREE.MathUtils.lerp(vel.x, targetSpeed, Math.min(1, accel * deltaTime / Math.max(1, Math.abs(targetSpeed)))),
      y: vel.y
    }, true);

    // Auto-flip sprite
    if (Math.abs(direction) > 0.01)
      this.actor.getComponent('SpriteRenderer')?.setFlipX(direction < 0);
  }

  jump() {
    const rb = this.actor.getComponent('RigidBody2D');
    if (this.jumpsRemaining > 0 || this.coyoteTimer > 0) this._doJump(rb);
    else this.jumpBufferTimer = this.properties.jumpBufferTime;
  }

  stopJump() {
    if (!this.properties.jumpCut) return;
    const rb = this.actor.getComponent('RigidBody2D');
    const vel = rb?.rigidBody.linvel();
    if (vel?.y > 0) rb.rigidBody.setLinvel({ x: vel.x, y: vel.y * 0.4 }, true);
  }

  _doJump(rb) {
    const vel = rb.rigidBody.linvel();
    rb.rigidBody.setLinvel({ x: vel.x, y: this.properties.jumpForce }, true);
    this.jumpsRemaining = Math.max(0, this.jumpsRemaining - 1);
    this.coyoteTimer = 0;
    this.jumpBufferTimer = 0;
  }
}

8. 2D Physics — Rapier2D
Use @dimforge/rapier2d-compat — a separate import from the 3D version. The two worlds are never active simultaneously.
javascriptimport RAPIER from '@dimforge/rapier2d-compat';

class Physics2DWorld {
  async init(settings = {}) {
    await RAPIER.init();
    this.world = new RAPIER.World(
      new RAPIER.Vector2(settings.gravity?.x ?? 0, settings.gravity?.y ?? -980)
    );
    this.pixelsPerUnit = settings.pixelsPerUnit || 100;
    this.bodyMap = new Map();
    this.eventQueue = new RAPIER.EventQueue(true);
    this.accumulator = 0;
    this.fixedStep = 1 / 60;
  }

  step(deltaTime) {
    this.accumulator += Math.min(deltaTime, 0.1);
    let steps = 0;
    while (this.accumulator >= this.fixedStep && steps < 8) {
      this.world.step(this.eventQueue);
      this.accumulator -= this.fixedStep;
      steps++;
    }
    this.syncToThreeJS();
    this.processEvents();
  }

  syncToThreeJS() {
    this.bodyMap.forEach(({ rigidBody, actor }) => {
      if (!rigidBody.isDynamic() && !rigidBody.isKinematic()) return;
      const pos = rigidBody.translation();
      actor.group.position.x = pos.x;
      actor.group.position.y = pos.y;
      actor.group.rotation.z = rigidBody.rotation();
    });
  }

  cleanup() {
    this.world.free();
    this.bodyMap.clear();
  }
}

  const start = context.getPinValue(node, 'Start');
  const end   = context.getPinValue(node, 'End');
  const debug = context.getPinValue(node, 'DrawDebug');

  const dir = { x: end.x - start.x, y: end.y - start.y };

  if (debug) {

      result.hit ? '#22c55e' : '#ef4444', 0.1);
    if (result.hit) debugDraw2D.drawCircle(result.point, 0.05, '#22c55e', 0.1);
  }

  context.setPinValue(node, 'ReturnValue', result.hit);
  context.setPinValue(node, 'OutHit', result);
  context.fireExec(node, result.hit ? 'Hit' : 'NoHit');
};

Performance Rules

Sprite UV update: only set uv.needsUpdate = true — never recreate geometry per frame
Tilemap rebuild: only rebuild changed cells on paint — not the full layer
Collision rebuild: debounced 200ms — not per-frame
Camera follow: use lerp in update() — never hard-set position per frame
Parallax: update material.map.offset only — no geometry changes
Physics debug overlay: depthTest: false, rebuild only when colliders change

System Isolation Rules

@dimforge/rapier2d-compat and @dimforge/rapier3d-compat are separate imports
2D world and 3D world are never active simultaneously
On scene mode switch: call physics2DWorld.cleanup() or physics3DWorld.cleanup() before init
All 2D actors live in their own Three.js Group — never mixed with 3D actor groups
Widget Blueprints, Blueprint graph canvas, and save/load are shared — do not fork them
