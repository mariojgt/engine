# What is Feather Engine?

Feather Engine is a TypeScript-based game engine/editor stack designed around Unreal-like mental models:

- **Editor-first workflow** with dockable tooling and asset managers.
- **Blueprint-style visual scripting** using node graphs.
- **Runtime lifecycle model** (`BeginPlay`, `Tick`, `OnDestroy`).
- **2D + 3D feature surface** including physics, AI, input, animation, and UI.
- **Web + desktop delivery** via Vite and Tauri.

## Core idea

Feather lets teams build gameplay through a hybrid model:

- visual graphs for designer speed,
- generated JavaScript for runtime execution,
- TypeScript modules for engine internals and advanced systems.

## Who this is for

- Teams that want a **single TypeScript codebase** for tools + runtime.
- Developers needing a **node-based gameplay layer** on top of code.
- Projects shipping to **browser and desktop shells**.
